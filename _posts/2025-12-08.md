# 4. GitHub에 푸시

1. 작성한 파일을 `_posts/` 폴더에 넣기  
2. GitHub Desktop이나 git CLI로 커밋  
3. 원격 저장소(`main` 또는 `master`)에 푸시  
4. GitHub Pages가 자동 빌드 후 블로그에 반영

---

# 5. 결과 확인

- 브라우저에서 블로그 접속  
- 다크모드 전환 → 글자 강조, 링크, 코드 블록 색상 확인  

---

즉, **마크다운으로 작성 → `_posts/`에 올리고 푸시 → GitHub Pages가 자동 배포**  
이 과정을 거치면 블로그 글이 올라가고, 다크모드 색상도 자동 적용됩니다.  

원하면 내가 **예시 글 + 다크모드 적용된 색상 샘플**을 만들어 바로 `_posts/`에 올릴 수 있게 보여드릴 수도 있습니다.
